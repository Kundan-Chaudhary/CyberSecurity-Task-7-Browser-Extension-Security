## 📌 Internship: Cyber Security Internship  
## 🎯 Objective:
Learn to identify and remove potentially harmful or unnecessary browser extensions to improve browser security.

---

# 🛠 Tools Used:
- Brave Browser (Chromium-based)
- Manual Security Review

---

# 📖 Task Description

In this task, I reviewed all installed browser extensions in Brave Browser to identify any suspicious, unused, or unnecessary extensions that could pose security risks.

---

# 🔍 Step-by-Step Process

## Step 1: Open Extension Manager
Opened Brave extension manager using:

brave://extensions/

Reviewed all installed extensions carefully.

---

## Step 2: Identified Installed Extensions

Initially Installed Extensions:

1. Adobe Acrobat: PDF edit, convert, sign tools  
2. Save image as Type  
3. WPS PDF – Read, Edit, Fill, Convert  

---

## Step 3: Security Review

- Found two PDF-related extensions installed.
- Having multiple extensions with similar functionality increases attack surface.
- Unused or redundant extensions may introduce security risks.

---

## Step 4: Removed Suspicious/Unnecessary Extension

❌ Removed Extension:
- WPS PDF – Read, Edit, Fill, Convert

### Reason for Removal:
- Duplicate functionality (already had Adobe Acrobat)
- Not actively used
- Reducing unnecessary browser permissions
- Minimizing security risks

---

# 🚨 How Malicious Extensions Can Harm Users

- Steal saved passwords
- Track browsing activity
- Capture keystrokes
- Inject malicious ads
- Redirect users to phishing websites
- Send sensitive data to external servers

---

# 🔐 Security Best Practices for Browser Extensions

- Install extensions only from official web stores
- Check reviews and ratings
- Review requested permissions before installing
- Remove unused extensions regularly
- Keep extensions updated
- Avoid installing duplicate tools

---
